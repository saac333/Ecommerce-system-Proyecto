# Sistema de Gestión Ecommerce - Proyecto POO
Evidencia de aprendizaje para la asignatura de Programación Orientada a Objetos.

## Estructura del Proyecto
El sistema está dividido en módulos independientes para cumplir con la **Encapsulación** (Unidad 3):
- **User:** Gestión y registro de usuarios.
- **Product:** Catálogo de productos con validaciones de precio.
- **Cart:** Manejo de carritos de compras por usuario.
- **Payment:** Procesamiento de pagos utilizando **Programación Funcional**.

## Cómo ejecutar
1. Abrir la terminal en la raíz del proyecto.
2. Ejecutar `go run main.go`.
3. El servidor estará disponible en `http://localhost:8080`.
